The message editor is the main tool you use to write effective, dynamic, and engaging messages for your customers.
It is used across Hellotext, from simple inbox replies to campaign messages and journey or playbook steps.

It has tools that let you enrich messages with attachments (images, videos, etc.), emojis, links, contact cards, and more.
This guide covers the basic formatting tools available in the editor.

### How to format your messages

The editor supports a variety of formatting options that allow you to create visually appealing messages. Here are some of the formatting options available:

#### Bold

To make a text section bold, select the text you want to format and use the shortcut:

- <kbd> Ctrl</kbd> + <kbd>B</kbd> on Windows
- <kbd>⌘ Command</kbd> + <kbd>B</kbd> on Mac

#### Italic

To make a text section italic, select the text you want to format and use the shortcut:

- <kbd> Ctrl</kbd> + <kbd>I</kbd> on Windows
- <kbd>⌘ Command</kbd> + <kbd>I</kbd> on Mac

#### Links

Links are created with the link tool in the editor. The link tool lets you add a tracked link inside your message.
The link element has the following icon <img style='display: inline;'  src="{{site.url}}/images/icons/link.svg">
and is located in the editor toolbar. You can open the link tool with these shortcuts:

- <kbd> Ctrl</kbd> + <kbd>K</kbd> on Windows
- <kbd>⌘ Command</kbd> + <kbd>K</kbd> on Mac
